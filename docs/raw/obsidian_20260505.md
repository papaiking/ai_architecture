# Obsidian

**Website**: https://obsidian.md/

Obsidian là một ứng dụng ghi chú và quản lý "second brain" dựa trên Markdown, giúp bạn xây dựng và tổ chức kho kiến thức cá nhân/dự án dưới dạng các tệp .md liên kết với nhau như một graph kiến thức.

## Obsidian dùng để làm gì?

- Ghi chú cá nhân, nhật ký, journal, daily notes cho công việc và cuộc sống
- Xây "knowledge base"/second brain cho dev, researcher, người học (lưu tài liệu, ý tưởng, trích dẫn, tài liệu nghiên cứu, meeting notes…)
- Quản lý dự án và công việc (project docs, roadmap, task list) nhờ hệ thống liên kết, tìm kiếm và plugin
- Viết bài, tài liệu kỹ thuật, blog draft bằng Markdown, có hỗ trợ LaTeX và code blocks

## Các tính năng chính

### 1. Markdown editor & file .md

- Hỗ trợ đầy đủ cú pháp Markdown: heading, list, table, bold/italic, link, image… trên nền file .md thuần, lưu ngay trên ổ đĩa của bạn
- Hỗ trợ LaTeX để viết công thức toán, và code blocks có syntax highlighting cho lập trình
- Vì là file .md nên bạn có thể đem sang dùng với các app khác (VS Code, Git, static site generator…)

### 2. Vault và tổ chức dữ liệu

- Mỗi "vault" là một thư mục trên máy, bên trong là hệ thống folder + file .md; bạn có thể tách vault cho cá nhân, công việc, side project, nghiên cứu
- Hỗ trợ nhiều vault song song, giúp tách biệt kiến thức giữa các mảng khác nhau
- Có sẵn "Help Vault" chứa tài liệu hướng dẫn ngay trong app

### 3. Link hai chiều và Graph View

- Cho phép tạo liên kết nội bộ giữa các note, với dạng link hai chiều (backlink) để thấy note nào đang trỏ đến note nào
- Graph View hiển thị toàn bộ note dưới dạng mạng lưới node–edge, giúp bạn visualize mối liên hệ, zoom, filter, tô màu cụm chủ đề

### 4. Tìm kiếm, tag, metadata

- Tìm kiếm toàn văn bản rất nhanh, hỗ trợ query nâng cao và filter theo nhiều tiêu chí
- Dùng tag và YAML frontmatter để thêm metadata (status, topic, project…) cho note, giúp tổ chức và truy vấn linh hoạt hơn

### 5. Các tiện ích ghi chú nâng cao

- **Outline**: hiển thị mục lục theo heading của note để dễ navigation
- **Daily notes**: tạo note mới theo ngày (YYYY-MM-DD) dùng cho journal, worklog, review
- **Random notes**: mở ngẫu nhiên một note trong vault để gợi ý ý tưởng hoặc ôn lại kiến thức cũ

### 6. Plugin và mở rộng

- Có hệ plugin rất phong phú (core + community plugin) để thêm tính năng như: kanban, calendar, task management, dataview, sync, publish…
- Cho phép tuỳ chỉnh giao diện, theme, hotkey, template để phù hợp workflow cá nhân của bạn

### 7. Offline, hiệu năng và quyền riêng tư

- Obsidian hoạt động offline, toàn bộ ghi chú nằm trên máy của bạn; tìm kiếm cũng thực hiện cục bộ nên rất nhanh
- Vì dữ liệu là file .md trên file system, bạn dễ dàng backup bằng Git, sync bằng Dropbox/iCloud/… hoặc dùng dịch vụ sync riêng của Obsidian nếu muốn
