**Organization**: langfuse
**Stars**: 25k+

## Overview

Open source LLM engineering platform: LLM Observability, metrics, evals, prompt management, playground, datasets. Integrates with OpenTelemetry, Langchain, OpenAI SDK, LiteLLM.

## Core Features

- **LLM Application Observability**: Track LLM calls and other logic in your app such as retrieval, embedding, or agent actions
- **Tracing**: Inspect and debug complex logs and user sessions
- **Metrics**: Usage statistics and performance monitoring
- **Evaluations**: LLM-as-Judge evaluation
- **Prompt Management**: Manage and version prompts
- **Playground**: Test prompts interactively
- **Datasets**: Create and manage test datasets
