# AnythingLLM

**Organization**: Mintplex-Labs
**Stars**: 58.7k
**License**: MIT

## Overview

The all-in-one AI application. Chat with your docs, use AI Agents, hyper-configurable, multi-user ready, and runs locally by default with zero setup.

## Key Features

- **Chat with Documents**: Ingest and chat with PDF, TXT, DOCX, and more
- **AI Agents**: No-code AI Agent builder with MCP compatibility
- **Multi-user Support**: Control access and experience per user
- **Custom Embeddable Chat Widget**: For your website
- **Full Developer API**: For custom integrations

## Supported LLM Providers

| Category | Providers |
|----------|-----------|
| Cloud | OpenAI, Anthropic, Google Gemini, Mistral, Groq, Cohere |
| Local | Ollama, LM Studio, LocalAI, llama.cpp |
| API | Azure OpenAI, AWS Bedrock, Hugging Face, DeepSeek, Moonshot |
| More | Together AI, Fireworks AI, Perplexity, OpenRouter, xAI, NVIDIA NIM |

## Embedder Models

- AnythingLLM Native (default)
- OpenAI, Azure OpenAI, Ollama, LM Studio, Cohere

## Vector Databases

- LanceDB (default)
- PGVector, Astra DB, Pinecone, Chroma, Weaviate, Qdrant, Milvus, Zilliz

## Deployment Options

- Docker (recommended)
- Desktop (Mac, Windows, Linux)
- AWS, GCP, Digital Ocean
- Render, Railway, Northflank
