# OptiLLM

## Overview

**OptiLLM** is an OpenAI API-compatible optimizing inference proxy that implements 20+ state-of-the-art techniques to dramatically improve LLM accuracy and performance on reasoning tasks - without requiring any model training or fine-tuning.

- **Org**: algorithmicsuperintelligence
- **GitHub**: https://github.com/algorithmicsuperintelligence/optillm
- **Stars**: 3.6k
- **License**: Apache-2.0
- **PyPI**: https://pypi.org/project/optillm/
- **HuggingFace Space**: https://huggingface.co/spaces/codelion/optillm

## Key Features

- Instant Improvements: 2-10x better accuracy on math, coding, and logical reasoning
- Drop-in Replacement: Works with any OpenAI-compatible API endpoint
- 20+ Optimization Techniques: From simple best-of-N to advanced MCTS and planning
- Zero Training Required: Just proxy your existing API calls through OptiLLM
- Production Ready: Used in production by companies and researchers worldwide
- Multi-Provider: Supports OpenAI, Anthropic, Google, Cerebras, and 100+ models via LiteLLM

## Implemented Techniques

- MARS (Multi-Agent Reasoning System) - Multi-agent reasoning with diverse temperature exploration
- CePO (Cerebras Planning and Optimization) - Combines Best of N, CoT, Self-Reflection, Self-Improvement
- CoT with Reflection - Chain-of-thought with thinking, reflection and output sections
- PlanSearch - Search algorithm over candidate plans for problem-solving
- ReRead (RE2) - Improves reasoning by processing queries twice
- Self-Consistency - Advanced self-consistency method
- Z3 Solver - Uses Z3 theorem prover for logical reasoning
- R* Algorithm - R* algorithm for problem-solving
- LEAP - Learns task-specific principles from few shot examples
- Round Trip Optimization (RTO) - Optimizes responses through round-trip process
- Best of N Sampling (BON) - Generates multiple responses and selects the best
- Mixture of Agents (MOA) - Combines responses from multiple critiques
- Monte Carlo Tree Search (MCTS) - MCTS for decision-making in chat responses
- PV Game - Prover-verifier game approach at inference time
- Deep Confidence - Confidence-guided reasoning with multiple intensity levels
- CoT Decoding - Chain-of-thought decoding without explicit prompting
- Entropy Decoding - Adaptive sampling based on token uncertainty
- AutoThink - Query complexity classification with steering vectors

## Plugins

- System Prompt Learning (SPL) - Model acquires program solving knowledge and strategies
- Deep Think - Gemini-like Deep Think using inference time scaling
- Long-Context CePO (LongCePO) - Planning and divide-and-conquer for infinite context
- Majority Voting - Selects most frequent answer from k candidates
- MCP Client - Model Context Protocol client for external tools
- Router - Routes requests to different approaches based on prompt
- Chain-of-Code (CoC) - CoT combined with code execution
- Memory - Short term memory for unbounded context length
- Privacy - Anonymize PII in requests
- Read URLs - Fetches URL content and adds to context
- Execute Code - Code interpreter for Python execution
- JSON - Structured outputs with outlines library
- GenSelect - Generative Solution Selection
- Web Search - Google searches with Selenium
- Deep Research - Test-Time Diffusion Deep Researcher
