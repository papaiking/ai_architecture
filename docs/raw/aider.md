# Aider - AI Pair Programming in Your Terminal

**Organization**: Aider-AI
**Stars**: 43k
**License**: MIT

## Overview

Aider lets you pair program with LLMs to start a new project or build on your existing codebase.

## Key Features

- **Cloud and local LLMs**: Works best with Claude 3.7 Sonnet, DeepSeek R1 & Chat V3, OpenAI o1, o3-mini & GPT-4o, but can connect to almost any LLM including local models
- **Maps your codebase**: Makes a map of your entire codebase for better understanding in larger projects
- **100+ code languages**: Python, JavaScript, Rust, Ruby, Go, C++, PHP, HTML, CSS, and more
- **Git integration**: Automatically commits changes with sensible commit messages
- **IDE integration**: Use from within your favorite IDE/editor
- **Images & web pages**: Add images and web pages to chat for visual context
- **Voice-to-code**: Speak with aider about your code
- **Linting & testing**: Automatically lint and test code on changes
- **Copy/paste to web chat**: Work with any LLM via web chat interface
