# Cline - Autonomous Coding Agent

**Organization**: cline
**Stars**: 59,627
**Website**: https://cline.bot

## Overview

Cline is an autonomous coding agent right in your IDE, capable of creating/editing files, executing commands, using the browser, and more with your permission every step of the way.

## Key Features

- **Autonomous coding**: Handles complex software development tasks step-by-step
- **File operations**: Create, edit, explore large projects
- **Terminal integration**: Execute commands directly in your terminal (VSCode v1.93+)
- **Browser automation**: Launch browser, click, type, scroll with Computer Use capability
- **MCP support**: Uses Model Context Protocol to create new tools
- **Human-in-the-loop**: Approve every file change and terminal command
- **Multi-platform**: VS Code, JetBrains, CLI, Cursor, Windsurf

## Capabilities

- Understand codebase structure
- Refactor large codebases
- Automate with CLI in scripts/cron jobs/CI pipelines
- End-to-end testing
- Interactive debugging

## Installation

5M+ installs across VS Code, JetBrains, and CLI.
