## What is Voicebox?

Voicebox is a **local-first AI voice studio** — a free and open-source alternative to ElevenLabs and WisprFlow in one app. Clone voices from a few seconds of audio, generate speech in 23 languages across 7 TTS engines, dictate into any text field with a global hotkey, and give any MCP-aware AI agent a voice of your choosing.

## Features

### Multi-Engine Voice Cloning

| Engine | Languages | Strengths |
|---|---|---|
| Qwen3-TTS (0.6B/1.7B) | 10 | High-quality multilingual cloning |
| Qwen CustomVoice | 10 | 9 curated preset voices |
| LuxTTS | English | Lightweight, 48kHz output |
| Chatterbox Multilingual | 23 | Broadest language coverage |
| Chatterbox Turbo | English | Paralinguistic emotion/sound tags |
| TADA (1B/3B) | 10 | HumeAI speech-language model |
| Kokoro | 8 | 50 preset voices, tiny 82M model |

### Voice Personalities

Attach a free-form personality to any voice profile, powered by a bundled Qwen3 LLM running locally. Agents can invoke the same modes over MCP.

### Agent Voice Output

One tool call (`voicebox.speak`) and any MCP-aware agent (Claude Code, Cursor, Cline) speaks to you in a voice you've cloned.

```ts
await voicebox.speak({
  text: "Deploy complete.",
  profile: "Morgan",
});
```

### Global Dictation

Hold a hotkey anywhere, speak, release — transcript pastes into the focused text field. LLM refinement optional.

### Stories Editor

Multi-track timeline for conversations, podcasts, and narratives.

### API & MCP Server

REST API at port 17493, built-in MCP server with tools: `voicebox.speak`, `voicebox.transcribe`, `voicebox.list_captures`, `voicebox.list_profiles`.

## Tech Stack

- **Desktop**: Tauri (Rust)
- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: FastAPI (Python)
- **TTS**: Qwen3-TTS, LuxTTS, Chatterbox, TADA, Kokoro
- **STT**: Whisper / Whisper Turbo
- **Local LLM**: Qwen3 (0.6B/1.7B/4B)
- **Inference**: MLX (Apple Silicon) / PyTorch (CUDA/ROCm/XPU/CPU)

## Platform Support

- macOS (Apple Silicon via MLX/Metal)
- Windows (CUDA, DirectML)
- Linux (ROCm)
- Intel Arc (IPEX/XPU)
- Docker
