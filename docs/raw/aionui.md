# AionUi

**Organization**: iOfficeAI
**Stars**: 22.3k
**License**: Apache-2.0

## Overview

A free, open-source, 24/7 Cowork app with AI Agents. Built-in agent with full file access, multi-agent support, and cross-platform (macOS/Windows/Linux).

## Key Features

- **Built-in Agent**: Zero setup, works immediately after install
- **Multi-Agent Mode**: Supports Claude Code, Codex, Gemini CLI, Qwen Code, Goose CLI, OpenClaw, and more
- **Office Assistants**: PPT, Word, Excel generation via Morph PPT and OfficeCli
- **Cross-Platform**: macOS / Windows / Linux
- **Remote Access**: WebUI + Telegram / Lark / DingTalk
- **Scheduled Automation**: Cron-based 24/7 unattended operation
- **Preview Panel**: 10+ format instant preview (PDF, Word, Excel, PPT, Code, Images)

## Supported AI Platforms

- **Cloud**: Gemini, OpenAI, Anthropic (Claude), DeepSeek, Qwen, Zhipu, Kimi, Baidu, Tencent
- **Local**: Ollama, LM Studio

## Comparison with Claude Cowork

| Feature | Claude Cowork | AionUi |
|---------|--------------|--------|
| OS | macOS only | macOS/Windows/Linux |
| Model | Claude only | 20+ platforms |
| Automation | Manual | Cron scheduled |
| Cost | $100/month | Free & Open Source |

## Use Cases

- File management & batch renaming
- Excel data processing & beautification
- Document generation (PPT, Word, Markdown)
- Scheduled 24/7 AI tasks
- Remote access from phone/browser
