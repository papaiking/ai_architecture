# Pipecat

## Overview

**Pipecat** is an open-source Python framework for building real-time voice and multimodal conversational agents. Orchestrate audio and video, AI services, different transports, and conversation pipelines effortlessly.

- **Org**: pipecat-ai
- **GitHub**: https://github.com/pipecat-ai/pipecat
- **Stars**: 12.1k
- **License**: BSD-2-Clause
- **Documentation**: https://docs.pipecat.ai
- **Website**: https://pipecat.ai

## What You Can Build

- Voice Assistants – natural, streaming conversations with AI
- AI Companions – coaches, meeting assistants, characters
- Multimodal Interfaces – voice, video, images, and more
- Interactive Storytelling – creative tools with generative media
- Business Agents – customer intake, support bots, guided flows
- Complex Dialog Systems – design logic with structured conversations

## Why Pipecat

- **Voice-first**: Integrates speech recognition, text-to-speech, and conversation handling
- **Pluggable**: Supports many AI services and tools
- **Composable Pipelines**: Build complex behavior from modular components
- **Real-Time**: Ultra-low latency interaction with different transports (e.g. WebSockets or WebRTC)

- **Speech-to-Text**: AssemblyAI, AWS, Azure, Cartesia, Deepgram, ElevenLabs, Google, Groq (Whisper), Mistral, NVIDIA, OpenAI (Whisper), Whisper, xAI, and more

- **Transport**: Daily (WebRTC), FastAPI Websocket, LiveKit (WebRTC), WebSocket Server, WhatsApp, Local
- **Video**: HeyGen, LemonSlice, Tavus, Simli
- **Memory**: mem0
- **Vision & Image**: fal, Google Imagen, Moondream
- **Audio Processing**: Silero VAD, Krisp Viva, Koala, ai-coustics, RNNoise
- **Analytics & Metrics**: OpenTelemetry, Sentry

## Pipecat Ecosystem

- **Pipecat Subagents** – Multi-agent systems with distributed pipelines and shared message bus
- **Pipecat Flows** – Structured conversations with complex state management
- **Pipecat CLI** – Create, monitor, and deploy projects
- **Whisker** – Real-time Pipecat debugger
- **Tail** – Terminal dashboard for Pipecat
- **Voice UI Kit** – Beautiful UI components for voice AI
- **Client SDKs**: JavaScript, React, React Native, Swift, Kotlin, C++, ESP32

## Getting Started

```bash
# Install uv
curl -LsSf https://astral.sh/uv/install.sh | sh

# Create new project
uv init my-pipecat-app
cd my-pipecat-app
uv add pipecat-ai

# Or with pip
pip install pipecat-ai
```
