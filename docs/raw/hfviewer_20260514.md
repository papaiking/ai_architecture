# HF Viewer — Visualize Any Hugging Face Model

**Org**: Embedl AB
**Website**: https://hfviewer.com/
**Date**: 2026-05-14
**Chrome Extension**: https://chromewebstore.google.com/detail/hugging-face-viewer/mmadlggmpkpiockpjfepaohcllbnakej

---

HF Viewer (hfviewer.com) is a free web tool that provides interactive architecture visualization for any Hugging Face model. Paste a Hugging Face URL or repo name and jump straight into a clear interactive model view.

## Key Features

- **Instant Visualization**: Paste a Hugging Face URL or repo ID to open an interactive model architecture graph directly in the browser, no installation needed
- **Granularity Levels**: Switch between overview and detailed traced blocks/paths
- **Model Family Comparison**: Family pages (e.g. Gemma 4) let you compare multiple related models with synchronized interaction
- **URL Magic**: Replace huggingface.co with hfviewer.com in any model URL to view it directly
- **Chrome Extension**: View each model directly from Hugging Face pages
- **Embeddable**: Press "Embed" to get code to embed visualizations in model cards
- **Interactive Blog**: Graph-to-text and text-to-graph loop for technical ML articles

## Popular Models Supported

- Gemma 4 family (E2B, E4B, 26B-A4B, 31B)
- DeepSeek V4 Pro
- YOLOv8n pose
- ViT base, CLIP
- GPT-2, T5
- Qwen 3.5/3.6 family
- Parakeet TDT speech recognizer
- And any other Hugging Face model

## Usage

1. Go to https://hfviewer.com/
2. Paste a Hugging Face URL or repo name (e.g., Qwen/Qwen3.5-4B)
3. Explore the interactive graph with zoom and granularity controls
4. Optionally embed visualizations in model cards or use the Chrome extension
