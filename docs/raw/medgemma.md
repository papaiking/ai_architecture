# MedGemma - Medical AI Model

**Source**: https://github.com/google-health/medgemma

## Overview

MedGemma is a collection of Gemma 3 variants trained for medical text and image comprehension. Healthcare-based AI applications builder.

## Details

- **Org**: Google-Health
- **Stars**: 1.4k
- **Forks**: 241
- **License**: Apache-2.0 (model: Health AI Developer Foundations License)

## Variants

1. **4B Multimodal** - SigLIP image encoder, chest X-rays, dermatology, ophthalmology, histopathology
2. **27B Text-only** - Medical text only

## Training Data

- Radiology images
- Histopathology patches  
- Ophthalmology images
- Dermatology images
- Medical text
