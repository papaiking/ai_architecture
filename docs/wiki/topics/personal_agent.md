# Personal Agents

Các tác nhân AI tập trung vào việc hỗ trợ cá nhân trong cuộc sống hàng ngày và xây dựng mối quan hệ lâu dài với người dùng.

## Agent Sources

- [AionUi](../sources/aionui.md) - Free, open-source 24/7 Cowork app (22.3k stars)
- [AnythingLLM](../sources/anything_llm.md) - All-in-one AI app with RAG (58.7k stars)
- [QwenPaw](../sources/qwenpaw.md) - Personal AI assistant (formerly CoPaw)
- [Hermes Agent](../sources/hermes_agent.md) - Self-improving agent with learning loop

## Agent skills
- [SkillClaw](../sources/skillclaw.md) - makes LLM agents progressively better by evolving reusable skills
