# Personal Agents

Các tác nhân AI tập trung vào việc hỗ trợ cá nhân trong cuộc sống hàng ngày và xây dựng mối quan hệ lâu dài với người dùng. Bao gồm các công cụ personal knowledge management và AI assistants.

## Personal Knowledge Management

- [Obsidian](../sources/obsidian.md) - Markdown-based "second brain" app with backlinks, Graph View, local-first, plugin ecosystem

## Agent Sources

- [AnythingLLM](../sources/anything_llm.md) - All-in-one AI app with RAG (58.7k stars)
- [AionUi](../sources/aionui.md) - Free, open-source 24/7 Cowork app (22.3k stars)
- [Hermes Agent](../sources/hermes_agent.md) - Self-improving agent with learning loop
- [QwenPaw](../sources/qwenpaw.md) - Personal AI assistant (16.4k stars, formerly CoPaw)
- [GBrain](../sources/gbrain.md) - AI agent memory & knowledge system by YC President Garry Tan (15.3k stars)
- [Personal AI Infrastructure (PAI)](../sources/personal_ai_infrastructure.md) — Life Operating System built on Claude Code: Pulse daemon, DA identity, Algorithm v6.3.0, 45 skills, by Daniel Miessler
