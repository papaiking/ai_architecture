# AionUi

**Organization**: iOfficeAI  
**Stars**: 22.3k  
**License**: Apache-2.0

## Overview

A free, open-source, 24/7 Cowork app with AI Agents. Built-in agent with full file access, multi-agent support, and cross-platform (macOS/Windows/Linux).

## Key Features

- **Built-in Agent**: Zero setup, works immediately after install
- **Multi-Agent Mode**: Claude Code, Codex, Gemini CLI, Qwen Code, Goose CLI, OpenClaw, 12+ more
- **Office Assistants**: PPT, Word, Excel generation via Morph PPT and OfficeCli
- **Cross-Platform**: macOS / Windows / Linux
- **Remote Access**: WebUI + Telegram / Lark / DingTalk
- **Scheduled Automation**: Cron-based 24/7 unattended operation
- **Preview Panel**: 10+ format instant preview

## Supported Platforms

- **Cloud**: Gemini, OpenAI, Anthropic, DeepSeek, Qwen, Zhipu, Kimi, Baidu, Tencent
- **Local**: Ollama, LM Studio
