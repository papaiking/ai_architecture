# MedGemma

**Type**: LLM - Medical AI

MedGemma is a collection of Gemma 3 variants trained for medical text and image comprehension.

## Overview

- **Org**: Google-Health
- **Stars**: 1.4k
- **License**: Apache-2.0

## Variants

| Model | Type | Description |
|-------|------|-------------|
| 4B | Multimodal | SigLIP encoder, medical imaging |
| 27B | Text-only | Medical text only |

## Capabilities

- Chest X-rays analysis
- Dermatology images
- Ophthalmology images  
- Histopathology slides
- Medical text comprehension
- Medical QA
