# AI Engineering from Scratch

**Type**: Comprehensive AI Learning Course

From linear algebra to autonomous agent swarms. Learn AI **with** AI, then ship the tools.

[GitHub](https://github.com/rohitg00/ai-engineering-from-scratch) | [Website](https://aiengineeringfromscratch.com)

## Overview

A comprehensive course with 260+ lessons across 20 phases (~290 hours). Every lesson produces something reusable: prompts, skills, agents, and MCP servers.

## Built-in Claude Code Skills

- **find-your-level** - 10-question quiz that maps your knowledge to a starting phase
- **check-understanding** - Per-phase quiz (8 questions) with feedback

## Course Output

Every lesson produces a reusable artifact:

- 📝 **Prompts** - Paste into any AI assistant
- 🎴 **Skills** - Install into Claude Code, Cursor
- 🤖 **Agents** - Deploy as autonomous workers
- 🔌 **MCP Servers** - Plug into any MCP-compatible AI app
