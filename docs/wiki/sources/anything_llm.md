# AnythingLLM

- **Organization**: Mintplex-Labs
- **Stars**: 58.7k

## Overview

The all-in-one AI application. Chat with your docs, use AI Agents, hyper-configurable, multi-user, and runs locally by default.

## Key Features

- **Chat with Documents**: PDF, TXT, DOCX, and more
- **AI Agents**: No-code builder + MCP support
- **Multi-user**: Permission control per user
- **Embeddable Chat Widget**: For websites
- **Vector Database**: LanceDB, PGVector, Pinecone, Chroma, etc.

## Supported LLMs

| Type | Examples |
|------|----------|
| Cloud | OpenAI, Anthropic, Gemini, Mistral |
| Local | Ollama, LM Studio, LocalAI |
| API | Azure, AWS Bedrock, DeepSeek, Moonshot |

## Installation

- **Docker**: Recommended
- **Desktop**: Mac, Windows, Linux
- **Cloud**: AWS, GCP, Digital Ocean, Railway
