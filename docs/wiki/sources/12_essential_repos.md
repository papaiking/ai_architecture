# 12 Essential Repos for ML/AI Engineers

Curated list of essential repositories covering ML basics, RAG, PyTorch, MLOps, and Agents.

## Overview

A collection of 13 repositories (including 1 bonus) designed for ML/AI engineers to learn and build AI applications. The list spans from beginner-friendly ML courses to advanced agent systems and MLOps tools.

## Repositories

### 1. [Machine Learning for Beginners (Microsoft)](https://github.com/microsoft/MLforBeginners)
- **Focus**: Basic ML concepts in Jupyter notebooks
- **Level**: Beginner-friendly and visual

### 2. [Learn PyTorch for Deep Learning](https://github.com/mrdbourke/pytorch-deep-learning)
- **Focus**: Learn PyTorch concepts
- **Level**: Beginner to intermediate

### 3. [Hands-On Large Language Models](https://github.com/HandsOnLLM/Hands-On-Large-Language-Models)
- **Focus**: Learn from basic language models to fine-tuning

### 4. [AI Agents for Beginners (Microsoft)](https://github.com/microsoft/ai-agents-for-beginners)
- **Focus**: Beginner-friendly AI agents course
- **Format**: Free 11-lesson hands-on curriculum

### 5. [Prompt Engineering Guide](https://github.com/brexhq/prompt-engineering)
- **Focus**: One-stop-shop for prompt engineering
- **Content**: Guides, papers, lectures, curated examples

### 6. [LLM Course](https://github.com/plnorris97/get-llm-course)
- **Focus**: Entire LLM lifecycle — design to deployment
- **Content**: Roadmaps + Colab notebooks

### 7. [GenAI Agents](https://github.com/hunkhill1/GenAIAgents)
- **Focus**: Building agent-based LLM systems
- **Content**: Simple tool-using agents to advanced workflows

### 8. [RAG Techniques](https://github.com/NirmalSamaran/rag-techniques)
- **Focus**: Comprehensive RAG tutorials collection

### 9. [Agents towards Production](https://github.com/Shubhamsaboo/awesome-llm-agents)
- **Focus**: Building blocks of GenAI agent stack

### 10. [Awesome Generative AI](https://github.com/stevenyg/awesome-generative-ai)
- **Focus**: GenAI tools, papers, interviews, repos
- **Use**: Personal R&D knowledge base

### 11. [Made With ML](https://github.com/GokuMohandas/Made-With-ML)
- **Focus**: Full ML product lifecycle
- **Content**: Design to CI/CD and monitoring

### 12. [Designing Machine Learning Systems](https://github.com/chanzuckerberg/idesign-ml)
- **Focus**: Summaries + code + diagrams from O'Reilly book
- **Use**: Architect real-world ML pipelines

### 13. [Jam with AI (Bonus)](https://github.com/Afank/jam-with-ai)
- **Focus**: Hands-on guides for production
- **Content**: RAG Systems, AI Agents, MLOps

## Key Topics Covered

- **ML Fundamentals**: Microsoft ML for Beginners, PyTorch, Hands-On LLMs
- **LLM Development**: LLM Course, Prompt Engineering, Made With ML
- **Agent Systems**: Microsoft AI Agents, GenAI Agents, Agents towards Production
- **RAG**: RAG Techniques, Jam with AI
- **MLOps**: Made With ML, Designing ML Systems, Jam with AI
