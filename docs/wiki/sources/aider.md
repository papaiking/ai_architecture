# Aider - AI Pair Programming

- **Organization**: Aider-AI
- **Stars**: 43k
- **License**: MIT
- **Website**: https://aider.chat/

## Overview

Aider lets you pair program with LLMs to start a new project or build on your existing codebase.

## Key Features

- **Cloud and local LLMs**: Claude 3.7 Sonnet, DeepSeek R1, OpenAI o1, GPT-4o, any LLM
- **Codebase mapping**: Maps entire codebase for larger projects
- **100+ languages**: Python, JavaScript, Rust, Go, C++, etc.
- **Git integration**: Auto-commits with sensible messages
- **IDE integration**: Use from your favorite editor
- **Images & web pages**: Add visual context to chat
- **Voice-to-code**: Speak to request changes
- **Auto lint/test**: Fix problems detected by linters and test suites

## Installation

```bash
pip install aider
```
