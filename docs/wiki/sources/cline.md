# Cline - Autonomous Coding Agent

**Organization**: cline
**Stars**: 59,627
**Website**: https://cline.bot

## Overview

Cline is an autonomous coding agent right in your IDE, capable of creating/editing files, executing commands, using the browser, and more with your permission every step of the way.

## Key Features

- **Autonomous coding**: Handles complex software development tasks step-by-step
- **File operations**: Create, edit, explore large projects
- **Terminal integration**: Execute commands directly in terminal
- **Browser automation**: Click, type, scroll with Computer Use capability
- **MCP support**: Model Context Protocol for extending capabilities
- **Human-in-the-loop**: Approve every file change and command

- VS Code (5M+ installs)
