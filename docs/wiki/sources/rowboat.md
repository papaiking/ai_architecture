# Rowboat — Open-Source AI Coworker with Memory

- **Source:** [github.com/rowboatlabs/rowboat](https://github.com/rowboatlabs/rowboat)
- **Author:** Rowboat Labs (Y Combinator S24)
- **Stars:** 14.1k | **License:** Apache-2.0


---

Rowboat is an **open-source AI coworker** that turns work into a knowledge graph and acts on it. It connects to email and meeting notes, builds long-lived memory, and helps you get work done — privately, on your machine.

![Rowboat Banner](../media/rowboat_banner.png)

## Key Features

- **Long-lived knowledge graph** — context accumulates over time, relationships are explicit
- **Obsidian-compatible Markdown vault** — transparent "working memory" you can inspect and edit
- **Local-first** — all data stored as plain Markdown, no proprietary formats
- **Live notes** — auto-updating notes (@rowboat) that track people, topics, competitors
- **MCP support** — extend with external tools (Exa, Slack, Linear, GitHub, etc.)
- **Bring your own model** — Ollama, LM Studio, or hosted APIs

## Integrations

- Gmail, Google Calendar, Rowboat meeting notes, Fireflies
- Composio.dev product integrations library
- MCP tools: Exa search, Twitter/X, ElevenLabs, Slack, Linear/Jira, GitHub

## Use Cases

- **Meeting prep** — pull past decisions, threads, open questions into a brief
- **Email drafting** — grounded in history and commitments
- **Docs & decks** — generate PDF slides from ongoing context
- **Follow-ups** — capture decisions, action items, owners
- **Voice memos** — record and auto-capture key takeaways into the graph
