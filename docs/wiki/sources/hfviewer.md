# HF Viewer

- **Source:** [hfviewer.com](https://hfviewer.com/)
- **Org:** Embedl AB

Interactive model architecture visualization for any Hugging Face model. Paste a URL, open an interactive graph in the browser — no installation needed.

[Website](https://hfviewer.com/) | [Chrome Extension](https://chromewebstore.google.com/detail/hugging-face-viewer/mmadlggmpkpiockpjfepaohcllbnakej)

## Overview

HF Viewer fills a gap in the Hugging Face ecosystem: a fast, general-purpose way to see how a model is put together. Instead of reading config files or staring at code, you get a visual map of the architecture with zoom from broad system shape down into specific substructure.

## Key Features

- **Instant Visualization** — Paste any HF URL or repo ID to open an interactive architecture graph
- **Granularity Control** — Switch between high-level overview and detailed traced blocks/paths
- **Family Comparison** — Compare related models (e.g. Gemma 4 family) with synchronized interaction
- **URL Magic** — Replace `huggingface.co` with `hfviewer.com` in any model URL
- **Chrome Extension** — View models directly from Hugging Face pages
- **Embeddable** — Embed visualizations in model cards via one-click embed code
- **Interactive Blog** — Graph-to-text and text-to-graph loop for technical ML articles

## Use Cases

- Understanding transformer architectures (decoder-only, encoder-decoder, MoE, multimodal)
- Comparing model families side by side
- Learning how models are put together without reading source code
- Teaching/education — embed in model cards or documentation
