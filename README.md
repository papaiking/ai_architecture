# hearme AI Wiki

A personal wiki for AI/LLM knowledge base built with MkDocs.

## Structure

```
docs/
├── raw/           # Raw source files (from URLs, repos)
├── wiki/
│   ├── topics/    # Topic overview pages
│   ├── sources   # Source summaries
│   ├── entities  # Entity pages (orgs, people)
│   ├── concepts # Concept pages
│   ├── media/   # Images & media
│   ├── index.md # Wiki index
│   └── log.md   # Activity log
```

## Topics

- **AI Fundamental** - Core AI/ML/LLM concepts
- **Building AI Applications** - Development & deployment
- **AI Agents** - Agent systems & skills

## Quick Start

```bash
# Install dependencies
pip install mkdocs mkdocs-material

# Serve locally
mkdocs serve

# Build
mkdocs build
```
